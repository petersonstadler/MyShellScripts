Auto-Shutdown

PT-BR

Autor: Peterson Siqueira Stadler

Este script foi criado com a intenção de monitorar o uso de recurso do sistema
e desliga-lo automáticamente se o uso for baixo. O objetivo é manter o meu
servidor de testes ligado somente quando estiver rodando algum programa.

EN-US

Author: Peterson Siqueira Stadler

This script was created with the intention of monitoring system resource usage
and automatically turn it off if usage is low. The goal is to keep my
test server turned on only when some program is running.
